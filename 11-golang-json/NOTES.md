### JSON
JSON : Javasciprt Object Notation

### Encode JSON
Encode JSON : Mengubah data menjadi JSON
- Marshal

### Decode JSON
Decode JSON : Mengubah JSON menjadi data
- Unmarshal

### JSON pada Golang
JSON object di Golang direpresentasikan dengan tipe data Struct

### JSON Array
JSON array di Golang direpresentasikan dengan tipe data Slice

### JSON Tag
JSON tag digunakan untuk memberikan informasi tambahan tentang field struct

### Map
JSON di Golang juga bisa direpresentasikan dengan tipe data map, map digunakan kasus data JSON yang dynamic atau tidak menentu fieldnya atau tidak tetap

### Streaming Decoder
Streaming decoder digunakan untuk mengubah JSON menjadi data secara stream
json.NewDecoder() && Decode()

### Streaming Encoder
Streaming encoder digunakan untuk mengubah data menjadi JSON secara stream
json.NewEncoder() && Encode()
